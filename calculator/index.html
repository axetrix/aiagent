Google Gen AI SDK Documentation - Submodules

This document provides an overview of the submodules available in the Google Gen AI SDK. It includes information about client, batches, caches, chats, files, live, models, tokens, tunings, and types modules.

Submodules

  • genai.client module: Contains AsyncClient and Client classes for making asynchronous and synchronous requests respectively.
  • genai.batches module: Includes AsyncBatches and Batches classes for creating and managing batch jobs.
  • genai.caches module: Provides AsyncCaches and Caches classes for managing cached content resources.
  • genai.chats module: Offers AsyncChats and Chats classes to create and manage chat sessions.
  • genai.files module: Contains AsyncFiles and Files classes to manage remotely stored files.
  • genai.live module: Includes AsyncLive class for connecting to the live server.
  • genai.models module: Provides AsyncModels and Models classes for various model-related operations, like computing tokens, generating content, and editing images.
  • genai.tokens module: Offers AsyncTokens and Tokens classes for creating auth tokens.
  • genai.tunings module: Includes AsyncTunings and Tunings classes for managing tuning jobs.
  • genai.types module: Defines pydantic models and typed dictionaries for various data structures used in the SDK.

For more detailed information, refer to the specific module documentation.
